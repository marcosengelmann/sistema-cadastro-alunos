# Atividade 2: Sistema de Cadastro de Alunos

[cite_start]Este projeto é uma aplicação web completa com CRUD de alunos usando Spring Boot e Thymeleaf[cite: 30, 31, 32].

## Como Executar
1. Certifique-se de ter o Java 25 instalado.
2. No terminal do projeto, execute:
   ```bash
   .\mvnw.cmd spring-boot:run```bash
